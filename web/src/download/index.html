Install it (allow “unknown sources”)

Because this app is delivered directly (not via the Play Store), Android asks you to approve installs from the app you used to open the file. One-time per source:

  1. Tap Download the app above on your phone — the .apk downloads.
  2. Open the downloaded file. Android prompts to allow installs from this source.
  3. Go to Settings → Apps → Special access → Install unknown apps, pick the browser or file app you used to open the APK, and turn on Allow from this source.
  4. Go back and tap Install.

On Android 8 and newer this permission is granted per app (per source). On older devices it’s a single toggle at Settings → Security → Unknown sources.

Updating

The app checks for a newer version when it launches and prompts you. When it does, come back to this page, download the latest APK, and install it over the top — your data and sign-in stay put.
